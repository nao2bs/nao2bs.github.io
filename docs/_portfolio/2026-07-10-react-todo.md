## プロジェクト概要

React と TypeScript を使用したシンプルで効率的な TODO 管理アプリです。

### 主な機能

- ✅ タスクの追加・編集・削除
- 📁 カテゴリ分類
- 🔔 優先度設定
- 💾 LocalStorage を使用した永続化
- 🎨 Tailwind CSS を使用したモダンな UI

### 技術スタック

- React 18
- TypeScript
- Tailwind CSS
- Zustand (状態管理)
