## プロジェクト概要

このプロジェクトは、Jekyllと Tailwind CSS を組み合わせた静的ポートフォリオサイトです。

### 技術スタック
- **静的サイト生成**: Jekyll
- **スタイリング**: Tailwind CSS
- **デプロイ**: GitHub Pages
- **コンテナ化**: Docker / Docker Compose

### 特徴

✨ **高速**: 静的サイトのため読み込み速度が非常に高速
🎨 **レスポンシブ**: Tailwind CSS により全デバイスに対応
🚀 **簡単デプロイ**: GitHub Pages で無料ホスティング
📝 **管理が簡単**: Markdown でコンテンツ管理

### コンポーネント

- ブログ一覧表示機能
- ポートフォリオ管理システム
- カテゴリー分類
- 日本語対応
